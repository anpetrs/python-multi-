  # Формула симметричной пирамиды

```
n = int(input())  
for i in range(n):  
    cur_cnt = (n // 2 + 1) - abs(n // 2 - i)  
    for j in range(cur_cnt):  
        print("*", end="")  
    print()
```

# Переставить min и max

```
s = []
for num in input().split():
    s.append(int(num))
mni, mxi = s.index(min(s)), s.index(max(s))
s[mni], s[mxi] = s[mxi], s[mni]
print(*s)
```
