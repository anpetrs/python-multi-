**Неявное преобразование**. Любое целое число (тип `int`) можно использовать там, где ожидается число с плавающей точкой (тип `float`), поскольку при необходимости Python автоматически преобразует целые числа в числа с плавающей точкой.  

**Число с плавающей точкой нельзя неявно преобразовать в целое число**. Для такого преобразования необходимо использовать **явное преобразование** с помощью команды `int()`. Преобразование чисел с плавающей точкой в целое производится с округлением в сторону нуля.  

**Преобразование чисел с плавающей точкой в целое** производится с округлением в сторону нуля, то есть `int(1.7) = 1`, `int(-1.7) = -1`.

# индексация  
если длина строки `s` равна `len(s)`, то при положительной нумерации слева направо, последний элемент имеет индекс равный `len(s) - 1`,  
а при отрицательной индексации справа налево, первый элемент имеет индекс равный `-len(s)`.

# Методы строк

| Метод | Описание |
|:---|:---|
| **Изменение регистра** | |
| `s.capitalize()` | Первый символ в верхний регистр, остальные в нижний. |
| `s.swapcase()` | Нижний регистр в верхний и наоборот. |
| `s.title()` | Первый символ каждого слова в верхний регистр. |
| `s.lower()` | Все символы в нижний регистр. |
| `s.upper()` | Все символы в верхний регистр. |
| **Поиск и проверка вхождений** | |
| `s.count(sub, start, end)` | Количество непересекающихся вхождений `sub` в `s[start:end]`. |
| `s.startswith(prefix, start, end)` | `True`, если строка начинается с `prefix`. |
| `s.endswith(suffix, start, end)` | `True`, если строка заканчивается на `suffix`. |
| `s.find(sub, start, end)` | Индекс первого вхождения `sub` или `-1`, если не найдено. |
| `s.rfind(sub, start, end)` | Индекс первого вхождения `sub`, поиск с конца строки. |
| `s.index(sub, start, end)` | Аналог `find`, но вызывает `ValueError`, если не найдено. |
| `s.rindex(sub, start, end)` | Аналог `index`, поиск с конца строки. |
| **Очистка и замена** | |
| `s.strip([chars])` | Удаляет пробелы (или `chars`) в начале и конце строки. |
| `s.lstrip([chars])` | Удаляет пробелы (или `chars`) в начале строки. |
| `s.rstrip([chars])` | Удаляет пробелы (или `chars`) в конце строки. |
| `s.replace(old, new)` | Заменяет все вхождения `old` на `new`. |
| **Проверка содержимого** | |
| `s.isalnum()` | `True`, если строка состоит только из букв и цифр. |
| `s.isalpha()` | `True`, если строка состоит только из букв. |
| `s.isdigit()` | `True`, если строка состоит только из цифр. |
| `s.islower()` | `True`, если все буквенные символы в нижнем регистре. |
| `s.isupper()` | `True`, если все буквенные символы в верхнем регистре. |
| `s.isspace()` | `True`, если строка состоит только из пробельных символов. |


> **Важно:** Методы `find()`, `index()`, `count()` и др. при использовании параметров `start` и `end` работают со срезом `s[start:end]`, но возвращают индекс относительно **всей** строки `s`.


| Метод | Что делает | Синтаксис | Пример | Результат | Важно |
|---|---|---|---|---|---|
| `split()` | Разбивает строку на части по разделителю , т.е. даёт список | `строка.split(разделитель)` | `"a,b,c".split(",")` | `['a', 'b', 'c']` | Если разделитель не указан, строка делится по пробелам |
| `join()` | Объединяет элементы списка в строку через разделитель | `"разделитель".join(список)` | `"-".join(["a", "b", "c"])` | `"a-b-c"` | Все элементы должны быть строками |

> **Важно:** Методы split() и join() являются строковыми методами, они не могут быть вызваны у списков. Строковый метод join() не работает для списков, у которых элементы не строкового типа данных.

# Методы списков

| Метод | Описание |
|:---|:---|
| `lst.insert(i, x)` | Вставляет элемент `x` в список под индексом `i`. |
| `lst.index(x, start, end)` | Возвращает индекс первого вхождения `x` в `lst[start:end]`. Если элемент не найден, вызывает `ValueError`. |
| `lst.remove(x)` | Удаляет первое вхождение элемента `x`. Если элемент не найден, вызывает `ValueError`. |
| `lst.pop(i)` | Удаляет и возвращает элемент с индексом `i`. Если индекс не указан, удаляет последний элемент. |
| `lst.reverse()` | Разворачивает список на месте. |
| `lst.count(x)` | Возвращает количество вхождений элемента `x` в списке. |
| `lst.clear()` | Удаляет все элементы списка. |
| `lst.copy()` | Создаёт поверхностную копию* списка. |
| `lst.sort(key=None, reverse=False)` | Сортирует список на месте. Можно задать функцию `key` и обратный порядок через `reverse=True`. |

> **Важно:** Методы `index()` и `count()` работают с элементами списка, а не со строками внутри него.  
> `sort()` и `reverse()` изменяют сам список и ничего не возвращают.  
> `copy()` создаёт поверхностную копию*: вложенные объекты не копируются отдельно.

>  *поверхностная копия копирует только первый уровень объекта. Всё, что вложено глубже, остаётся общим.
